Language filtering

Installation

bower install azachar/angular-language-picker --save

  • Add the modul

    'language-picker'

    to your main angular module. 

    
var module = angular.module('myApp',[
 		'language-picker'
]);

  • Add the picker

    to your html file

    <language-picker
	languages="['en-US', 'fr-FR']"
	on-change="someCallback(language)">
 Choose a language
</language-picker>

  • Process the selection

    in your controller

    
module.controller('MyCtrl', function ($scope) {

  $scope.onLanguageChange = 
   function (language) {
     //process the language change
    $scope.myLang = language.code;
  };

});

  • Add dependencies

    If you are using grunt-wiredep, it should be done automatically.

    Otherwise ensure that you have the css and js dependencies for AngularJs, Bootstrap, UI-Bootstrap, FontAwesome, Flag-Icon and this Angular Language Picker at your index.html. 

    <!-- ... -->
<head>
 <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.2.0/css/bootstrap.min.css">
 <link rel="stylesheet" type="text/css" href="//cdnjs.cloudflare.com/ajax/libs/font-awesome/4.2.0/css/font-awesome.min.css">
 <link rel="stylesheet" href="http://lipis.github.io/flag-icon-css/css/flag-icon.min.css">

<!-- ...  end of the body element-->
 <script src="//ajax.googleapis.com/ajax/libs/angularjs/1.2.25/angular.min.js"></script>

</body>

Attributes

The language-picker markup has the following attributes:

{{codeSamples.attributes}}
  • languages (array): A list of all supported language codes.
  • on-change (function): A callback. Should take a single argument, language, which is the country code of the language that was selected.
    The language object has the following fields:

    Example for locale sk-SK
    • language.englishName= Slovak (Slovakia)
    • language.nativeName= Slovenƒçina (Slovakia)
    • language.code= sk-SK
    • language.asLowDashCode()= sk_SK
    • language.country= sk

  • flags (boolean): Enable or disable displaying language flag next a language. The flags are enabled by default. The country flags are parsed from locale. So please specify a locale fully - 'en-US' instead of 'en'.
  • template (String): a name for a custom modal template. The default value is language-picker-dialog.tpl.html
